PASS_BY_VALUE
The rustc_pass_by_value lint marks a type with #[rustc_pass_by_value] requiring it to always be passed by value. This is usually used for types that are thin wrappers around references, so there is no benefit to an extra layer of indirection. (Example: Ty which is a reference to an Interned<TyS>)
