Expand description

Indent each line according to the specified indent. e.g.

ⓘ
foo!{
x,
y,
foo(
    a,
    b,
    c,
),
}

will become

ⓘ
foo!{
    x,
    y,
    foo(
        a,
        b,
        c,
    ),
}
