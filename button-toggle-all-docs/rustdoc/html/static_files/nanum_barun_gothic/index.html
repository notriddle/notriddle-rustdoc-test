Files related to the Nanum Barun Gothic font.

These files are used to avoid some legacy CJK serif fonts in Windows.

Note that the Noto Sans KR font, which was used previously but was not very readable on Windows, has been replaced by the Nanum Barun Gothic font. This is due to Windowsâ€™ implementation of font rendering that distorts OpenType fonts too much.

The font files were generated with these commands:

pyftsubset NanumBarunGothic.ttf \
--unicodes=U+AC00-D7AF,U+1100-11FF,U+3130-318F,U+A960-A97F,U+D7B0-D7FF \
--output-file=NanumBarunGothic.ttf.woff2 --flavor=woff2
