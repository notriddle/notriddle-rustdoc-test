struct GeneratorSavedLocals(BitSet<Local>);
Expand description

The set of Locals that must be saved across yield points.

GeneratorSavedLocal is indexed in terms of the elements in this set; i.e. GeneratorSavedLocal::new(1) corresponds to the second local included in this set.
